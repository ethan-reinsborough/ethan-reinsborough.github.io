Instructions

How to Install FTB Infinity Evolved (With Extra Mods)

1. Download CurseForge desktop application.
2. Download FTB Infinity Evolved
3. Check your settings to allocate enough memory (RAM), around 4-10 GB is good. Follow these final steps below

How to Join a Minecraft Server

1. Open Minecraft and go to Multiplayer.
2. Add a new server with the IP: mc.example.com.
3. Click "Join Server" and start playing!

Basic Commands

/home - Teleport to your home.
/sethome [name of home] - Set a home.
/back - Return to previous tp or death point.
